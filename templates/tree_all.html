Decision Tree for Non-correlated Features

The above image is the decision tree created based on a subset of the data using sklearn's DecisionTreeClassifier.

Due to the high correlation between radius, perimeter, and area only one should be used as a feature; likewise for compactness, concavity, and concavepoint.

The desion tree first splits on x[13] which is perimeter_worst and has a gini score that seems reasonible. This means an importance is placed the perimeter_worst value, which is the largest perimeter length of a cell taken from a tumor. The decision tree had a score of 0.929, which is seems to be overfitted.



Random Forest for Non-correlated Features

Using sklearn's RandomForestClassifier, a random forest created based the same data subset using 200 trees to make a prediction. The RandomForestClassifier also placed importance on the perimeter_mean and scored 0.989. This score seems to imply overfitting of the data and other techniques could be used to rectify this problem.
