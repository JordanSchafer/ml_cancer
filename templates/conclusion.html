Conclusion

Decision Trees and Random Forests

The decision trees and random forests scored fairly well. The decision tree created using the mean values subset had it's first split on perimeter_mean, which is the average cell perimeter, implying a large importance in of that value. The random forest created using the same subset, also placed importance on perimeter_mean.

The decision tree using the noncorrelated subset had it's first split on perimeter_worst, which is the largest cell perimeter, implying a large importance in that value. This same importance is refelected in the random forest for the same data subset.

Though both random forests could be overfitting to the data and boosting algorithms could be used to help offset this issue.

K Nearest Neighbors

Both unscaled and scaled knn scored well. Using the elbow method to determine how many neighbors to use, an appropriate number of neighbors appears to be 7 for both.
